
-git config

    ** mostrará todas las configuraciones que tiene git.

-git config --list --show-origin

    ** mostrará donde esta ubicado el archivo de configuración de git.

-git config --global user.name "Nombre de Git"

    ** cambia la configuracion global del usuario git

-git config --global user.email "emaildegit@git.com"

    ** cambia la configuración global del email del usuario git

-git config --list

    ** visualizar configuraciones

-git remote add Nombre_variable_referirse_repositorio URL_REPOSITORIO

    ** Sirve para añadir otra nueva variable remote con un repositorio diferente
