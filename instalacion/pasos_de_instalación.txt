---- LINUX -------

    *** Ingresar al command-line ***

        - Si se tiene distribuciones derivabas de debian como ( Ubuntu ) verficar si se tiene el comando

            -apt-get ( ingresarlo en el command-line )

            -Instalar git a través del sistema operativo.

                -Primero se debe actualizar la lista de paquetes de linux ( recordar tener permisos de Admin )

                    -sudo apt-get update

                    ( Sudo es para que tenga permisos de administrador nuestro comando )
                
                -Instalar git con el comando apt-get

                    -apt-get install git
